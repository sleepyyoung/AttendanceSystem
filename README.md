# AttendanceSystem
**基于 Django + (tensorflow)keras/face-recognition 【python3.6】的人脸识别签到系统**

**建议使用Anaconda新建一个python3.6的虚拟环境进行操作**

```cmd
conda create -n your_env_name python=3.6
conda activate your_env_name
pip install requirements.txt
```
